Where should I contact for Billing related problem?

We would love to handle all such issues but Apple handle all the billing related issues.

Where should I contact for technical problem?

If any of the FAQ's don't answer your questions, please feel free to email us at ipooja@mediaagility.com

If I have purchased an application and I have upgraded my device, will that application be removed?

No,the application should not be removed. In case the application is removed, you can again download the application from the store and you will not be charged for the same.

If I have updated the application with new version, what will happen to the packages already purchased?

The packages will not be removed from the application. You may see an option to restore your earlier purchases based on the kind of update we do.

If I have deleted the application, do I need to again purchase the packages?

No,the packages can be restored using the restore functionality. You will not be charged for the same as Apple keeps take care of that. For restoring the packages, Click on Get More Button, then click on the package to be restored. Click on Restore button.
